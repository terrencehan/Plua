Plua 
=========================
An simple implementation of Lua 5.1 written with pure Perl, it is also a part of my graduation project.

DEPENDENCIES
------------------------

This module requires these other modules and libraries:

* Moose
* MooseX::Declare
* MooseX::ClassAttribute
